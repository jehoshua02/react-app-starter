# React Scaffold

Couldn't find a Yeoman generator with webpack, react, reflux and react-router.


## Setup

```shell
npm install
```


## Tests

```shell
npm test
```

Coverage reports saved to `coverage/`.


## Build

```shell
npm run build
```

Build output saved in `public/`.


## Watch

```shell
npm run watch
```


## Preview

```shell
open public/index.html
```


## Serve

```shell
npm run serve
```
