# React Scaffold

Couldn't find a Yeoman generator with webpack, react, reflux and react-router.


## Setup

```shell
npm install
```


## Test

```shell
npm test
```

Coverage reports saved to `coverage/`.


## Serve

```shell
npm run serve
```


## Watch

```shell
npm run watch
```


## Build

```shell
npm run build
```

Build output saved in `public/`.


## Preview

```shell
npm run preview
```
